== Arbor


== Development

Start the nREPLs and build everything for debug.

----
$ clj -M:dev:shadow-cljs watch app
----

Connect to the REPL:

----
$ clj -T:repl
----

Run tests:

[source,clojure]
----
user=> (refresh) (run-tests)
----

Testing on tablet:

----
$ clj -M:shadow-cljs release app
$ npx cap run android
----
